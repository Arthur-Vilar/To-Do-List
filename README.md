# To-Do List | Python + SQLite3

Projeto desenvolvido com o objetivo de praticar operações CRUD (Create, Read, Update, Delete), persistência de dados em banco SQLite e organização modular do código.

A aplicação permite o gerenciamento de tarefas por meio de um menu interativo via terminal e encontra-se em processo de implementação de interface web utilizando Flask.

## Funcionalidades

- Operações CRUD completas
- Persistência de dados com SQLite
- Validação de entradas
- Menu interativo via terminal

## Próximas Implementações

- Interface web com Flask
- Melhorias na estrutura da aplicação
- Aperfeiçoamento da experiência do usuário
